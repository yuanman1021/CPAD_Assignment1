## Project Description

This project is an Employee Directory web application built using Vue 3, Axios, Express, Node.js and MySQL.

The system allows users to add, view, search, sort, edit and delete employee records. Employee data is stored permanently in a MySQL database.

## Technologies Used

Vue 3, Vite, Axios, Node.js, Express, MySQL, mysql2, Laragon

## Setup Instructions

1. Start Laragon and make sure MySQL is running.

2. Open HeidiSQL and run: sql/schema.sql

3. Open terminal 1 and run: "npm run server". (run on localhost 3001 port)

4. Open terminal 2 and run: "npm run dev". (run on localhost 5174 port)
